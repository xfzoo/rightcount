RightCount

A minimal Chrome extension for instant right-click text counting.

RightCount shows a lightweight count bubble when you right-click selected text on a web page.

  • Chinese is counted by character
  • English is counted by word
  • Numbers are counted as one token
  • Punctuation is ignored

Everything runs locally in the browser. Selected text is not stored or sent to any server.
