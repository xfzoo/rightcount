Privacy Policy for RightCount

Last updated: 2026-03-30

RightCount is a lightweight extension that counts selected text on web pages and shows the result in a small bubble inside the browser.

What data the extension accesses

The extension only accesses the text that you actively select on the current web page when you right-click.

How the data is used

The selected text is used only to calculate a count and show that result to you.

What the extension does not do

  • It does not send selected text to any server.
  • It does not store selected text after the count is shown.
  • It does not collect analytics.
  • It does not track browsing history.
  • It does not sell or share user data.

Data storage

The extension does not store personal data or selected text.

Third-party sharing

No user data is shared with any third party.
